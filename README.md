This is my personal project around Kalman Filter Theory

The goal is to estimate states of a noisy system in order to have a better control.

Several obserever, such as Luenberger observer, and differents Kalman filter type will be implemented.

Several controllers such as PID, RST, LQ, Hinf, pole placement with matrix feedback gain, and Lyapunov theory for feedback gain.
Internal model to reject non random disturbance could also be implemented.
